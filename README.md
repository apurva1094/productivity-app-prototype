# 📚 StudyFlow – Productivity App Prototype
Prototype for a productivity app built during **FOSS Hack 2026**.

StudyFlow helps students manage tasks, stay focused using the Pomodoro technique, and track productivity in a simple interface.

---

## 🎯 Problem Statement
Students often juggle assignments, exams, and daily goals across multiple tools.
This leads to missed deadlines, reduced focus, and poor organization.

---

## 💡 Proposed Solution
StudyFlow is a lightweight productivity application that combines:

- 📋 Task management
- ⏱ Pomodoro focus timer
- 🔎 Task search and filtering
- 🎯 Priority tagging (High / Medium / Low)
- 📅 Due date tracking
- ⚠️ Overdue task detection
- 📊 Productivity analytics
- 🌙 Dark / Light mode

The goal is to create a **simple, distraction-free productivity tool for students**.

---

## 🚀 Current Features

- Add, edit, delete tasks
- Mark tasks as completed
- Search tasks
- Filter tasks (All / Done / Pending / Overdue)
- Priority-based sorting
- Progress tracking bar
- Focus score calculation
- Pomodoro timer
- Pomodoro streak tracking
- Milestone celebration with confetti

---

## 👥 Target Users

- College students
- School students
- Self-learners
- Anyone who wants a lightweight productivity system

---

## 🛠 Tech Stack

- React
- JavaScript
- CSS
- LocalStorage (for saving tasks and streak data)
